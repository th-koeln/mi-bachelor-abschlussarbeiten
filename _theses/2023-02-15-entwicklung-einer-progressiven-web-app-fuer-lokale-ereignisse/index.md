Progressive Web Apps (PWAs) revolutionieren die Art, wie wir mobile Anwendungen entwickeln und nutzen. Diese Bachelorarbeit befasst sich mit der Entwicklung einer PWA zur Entdeckung und Verwaltung lokaler Ereignisse und Events.

Die Anwendung kombiniert die Vorteile nativer Apps mit der Flexibilität von Webanwendungen. Durch Service Worker ermöglicht sie Offline-Funktionalität, während Push-Benachrichtigungen Nutzer über neue Events in ihrer Nähe informieren.

Ein besonderer Fokus liegt auf der Performance-Optimierung und der nahtlosen User Experience. Die App nutzt moderne Web-APIs wie die Geolocation API und IndexedDB für lokale Datenspeicherung. Das responsive Design gewährleistet eine optimale Darstellung auf verschiedenen Endgeräten.

Die Evaluierung zeigt, dass PWAs eine kosteneffiziente Alternative zu nativen Apps darstellen können, insbesondere für location-based Services wie Event-Discovery-Plattformen.
