Voice User Interfaces (VUIs) werden zunehmend in alltäglichen Anwendungen eingesetzt, jedoch fehlen etablierte Usability Testing Methoden für sprachbasierte Interfaces. Diese Bachelorarbeit entwickelt systematische Ansätze für das Testen von VUIs.

Untersucht wurden verschiedene Testmethoden: Think-Aloud-Protokolle, Wizard-of-Oz-Tests, A/B-Tests für Sprachausgaben und automatisierte Tests mit synthetischen Sprachdaten. Entwickelt wurde ein Toolkit für VUI-Usability-Tests mit Integration gängiger Sprachassistenten.

Die Studie analysierte typische Usability-Probleme in VUIs: Missverständnisse bei der Spracherkennung, unklare Nutzerführung, fehlende Error-Recovery-Mechanismen und inadäquate Rückmeldungen. Daraus wurden Design Guidelines für conversational interfaces abgeleitet.

Eine empirische Evaluation mit 60 Probanden validierte die entwickelten Testmethoden anhand von drei Alexa Skills. Die Ergebnisse zeigen, dass spezialisierte VUI-Testverfahren signifikant mehr usability issues identifizieren als herkömmliche GUI-Testmethoden.
